# Portfolio Adventure Game

An interactive 2D platformer portfolio website built with Phaser 3 and Vite.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser to [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` folder, ready for deployment to Vercel or any static hosting service.

## Project Structure

```
portfolio-game/
├── src/
│   ├── config/          # Phaser game configuration
│   ├── scenes/          # Phaser game scenes
│   ├── assets/          # Asset documentation
│   ├── main.ts          # Entry point
│   └── style.css        # Global styles
├── public/
│   ├── assets/          # Game assets (images, audio)
│   └── RESUME_HERE.pdf  # Resume PDF (replace with yours)
└── index.html           # Landing page HTML
```
